  • justify-content được dùng để căn chỉnh phần tử theo trục main axix (mặc định main axis sẽ nằm ngang): lưu ý các value flex-start, start, flex-end, end => các value có tiền tố là flex thì sẽ căn chỉnh cố định theo chiều của main axis (flex-start bên trái và flex-end bên phải). Còn 2 thuộc tính start và end thì sẽ tùy thuộc vào ngôn ngữ của hệ thống, nếu ngôn ngữ của hệ thống bắt đầu đọc từ bên phải thì bên phải sẽ là start
  • align-items được dùng để căn chỉnh phần tử theo trục cross axis (mặc định thì cross axis sẽ năm dọc) Lưu ý value stretch => value stretch sẽ kéo dãn phần tử con sao cho chiều cao của phần tử bằng chiều cao của phần content

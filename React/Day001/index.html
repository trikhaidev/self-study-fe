  • Không dùng được lambda expression cho export default
  • không thể import export default theo kiểu {}
  • khi import *, chúng ta hoàn toàn có thể sử dụng export default, tuy nhiên tên của export default sẽ là default
  • import defaultExport,{} from "./service.js"; => defaultExport bắt buộc phải viết trước
  • Khi một biến được import từ một file khác thì nó sẽ không thể bị thay đổi giá trị
  • Khi nào quá trình import hoàn tất thì các câu lệnh của file đó mới được thực thi. Trong quá trình import thì các câu lệnh trong file export sẽ được thực thi trước (định nghĩa, khai báo các method, biến, class,...). Tuy nhiên, không vì thế mà file export sẽ chạy trước file import, file import vẫn sẽ chạy trước vì nếu k chạy thì sao câu lệnh import đc thực thi. Lưu ý, chỉ khi quá trình import hoàn tất thì các câu lệnh của file import mới được thực thi
